# VirtualList

**正在开发。**

VirtualList是一个配置极为简单的文件共享服务器。

特点：

- 默认配置即可部署，只需更改一个文件并启用HTTPS反向代理即可安全地公网访问
- 无维护需求
- 程序、数据库和文件内容可被直接打包带走
